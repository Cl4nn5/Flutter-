### Types of Widgets
  1. Stateless Widget 
    * 이전 상호작용의 어떠한 값도 저장하지 않음
  2. Stateful Widget  
    * Value 값을 지속적으로 추적 & 보존
  3. Inherited Widget
  > flutter는 트리구조의 widget으로 구성되어있음
### Flutter에서 Class 와 Widget
  1. Class  
     * 객체로서 가져야 할 "속성"과 "기능"을 정의 - 설계도
  2. 객체  
     * 클래스가 정의된 후 메모리상에 할당 되었을 때
  3. 인스턴스  
     * 속성과 기능을 똑같이 가진 클래스를 기반으로 생성되어 프로그래밍상에서 사용되는 대상
